# Window Tiler

Windows 환경에서 모니터 화면을 지정된 비율로 세로/가로 분할하고, 등록된 창(Window)들을 각 영역에 자동으로 배치해 주는 타일링(Tiling) 유틸리티입니다. 복잡한 모니터 화면을 깔끔하게 유지할 수 있도록 도와줍니다.

## 주요 기능

### 1. 직관적인 2단 패널 GUI (설정 및 미리보기)
- **모니터 선택**: 다중 모니터 환경을 지원하며, 상단 콤보박스에서 작업할 모니터를 선택할 수 있습니다.
- **실시간 분할 미리보기 (Canvas)**: 좌측 하단의 캔버스 화면을 통해 현재 모니터의 실제 종횡비가 반영된 분할 상태를 실시간으로 확인할 수 있습니다.
- **마우스 드래그 분할 조절**: 캔버스 상의 점선(분할선)을 마우스로 직접 드래그하여 직관적으로 화면 비율을 조정할 수 있습니다.
- **수치 기반 정밀 조정**: 캔버스 아래의 입력란을 통해 0~100% 사이의 비율을 숫자로 정밀하게 입력하여 분할선을 추가/제거할 수 있습니다.

### 2. 강력한 창 배정 방식 (슬롯 할당)
- **자동 지정**: 버튼 하나로 현재 화면에 띄워져 있는 창들을 인식하여 빈 분할 영역(슬롯)에 순차적으로 자동 할당합니다.
- **선택 자동 지정**: 목록에서 원하는 창을 직접 선택하여 자동으로 배정합니다.
- **간격(Gap) 조절**: 창과 창 사이의 여백(Gap) 픽셀 크기를 실시간으로 조절할 수 있습니다. 윈도우11의 보이지 않는 그림자 테두리까지 계산하여 완벽하게 밀착시킵니다.
- **드래그 앤 드롭 스왑**: '창 배정 기록'에서 항목을 드래그하여 다른 슬롯으로 이동하면 창이 스왑됩니다.

### 3. 안전한 윈도우 교체 (투명 덮개 기술)
- 가장 넓은 영역을 차지하는 '메인 슬롯'과 나머지 '사이드 슬롯'으로 구분됩니다.
- 타일링이 작동 중일 때, **사이드 슬롯 영역에는 보이지 않는 투명한 덮개(Overlay)가 씌워집니다.**
- 사용자가 사이드 슬롯의 창을 클릭하면, 실수로 내부의 링크나 버튼이 눌리는 것을 방지하고 해당 창을 즉시 '메인 슬롯'의 위치로 교환(Swap)해 줍니다.

### 4. 고정 및 덮개 토글 기능
- **고정**: 특정 슬롯을 고정하면 메인 슬롯과 스왑되지 않습니다. '고정' 컬럼을 더블클릭하거나 우클릭하여 설정하세요.
- **덮개 토글**: 덮개 표시를 켜거나 끌 수 있습니다. '덮개' 컬럼 더블클릭으로 토글하세요.
- 고정된 슬롯은 드래그 앤 드롭으로도 스왑되지 않습니다.

### 5. 프로필 프리셋 관리
- **프로필 저장/변경**: 내가 설정한 완벽한 화면 분할 비율을 이름과 함께 프로필로 저장하고 언제든 불러올 수 있습니다.
- **기본 프로필 제공**: `기본(3분할)`, `좌우 2분할`, `균형(4분할)` 등의 프리셋이 내장되어 있습니다.

### 6. 백그라운드 작동 및 전역 단축키
- GUI 설정 창을 닫으면 트레이로 이동할지 완전 종료할지 선택합니다.
- **트레이 아이콘 제어**: 우클릭하여 일시정지, 재개, 설정 창 호출, 완전 종료를 제어할 수 있습니다.
- **전역 단축키**: 다른 게임이나 작업을 하는 도중에도 단축키를 눌러 언제든 타일링 기능을 끄거나 켤 수 있습니다.

## 기술 스택

- **Language**: Python 3.10+
- **GUI**: Tkinter (tk)
- **Window Management**: pywin32 (win32gui, win32con)
- **Tray**: pystray
- **Package Manager**: UV

```
win_tiler/
├── src/

│   ├── tiling_engine.py    # 타일링 핵심 로직
│   ├── overlay_manager.py   # 투명 덮개 관리
│   ├── tray_manager.py      # 시스템 트레이 관리

│   │   ├── slot_tree.py     # 창 배정 기록 Treeview
│   │   └── window_selector.py # 창 선택 다이얼로그
│   └── ...
├── README.md
└── pyproject.toml
```

## 시작하기

### 사전 요구사항
- Python 3.10 이상
- Windows 환경

### 설치 및 실행

UV(Python 패키지 매니저)를 사용하여 실행 환경을 세팅하고 실행할 수 있습니다.

```bash
# 개발 서버 실행
uv run src/main.py
```

## 데이터 저장소

설정 데이터는 프로젝트 루트 폴더에 JSON 형식으로 저장됩니다.

| 파일 | 설명 |
|------|------|
| `config.json` | 마지막 사용 모니터, 단축키, 창 간격(Gap), 슬롯별 창 배정 기록 |
| `profiles.json` | 사용자가 추가/저장한 화면 분할 비율 프리셋 목록 |
